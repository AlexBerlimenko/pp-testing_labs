Про цей проект

Це простий HTML-файл, створений для демонстрації роботи з системою контролю версій Git.

Мета роботи

Закріпити практичні навички роботи з системою Git та навчитися створювати прості веб-сторінки.

Завдання

  • Встановити Git та створити локальний репозиторій.
  • Зробити перший коміт та завантажити проєкт на GitHub.
  • Відредагувати HTML-файл і зробити повторний коміт.

Результати

У результаті виконання лабораторної роботи студент отримав практичні навички роботи з Git та створення HTML-документів.
